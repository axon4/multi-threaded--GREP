# Multi-Threaded GREP

* search for text recursively within a folder
* utilise power of concurrency in Go to quickly find the queried text
* print results to terminal like GREP (Global--Regular-Expression---Print)

## Command-Line--InterFace

* `go run ./ [query] [folder]`
